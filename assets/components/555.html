Functional Diagram

Astable Operation

T = 0.7 × (R1 + 2R2) × C1

Mark (time on) = 0.7 x (R1 + R2) x C1

Space (time off) = 0.7 x R1 x C1

T = time in seconds
R1,R2 = resistance in ohms
C1 = capacitance in farads

Monostable Operation

T = 1.1 × R1 × C1

T = time period in seconds
R1 = resistance in ohms
C1 = capacitance in farads
